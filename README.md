# AES-CBC-ECB-Example
Image encryption script used to show the difference between the two algorithms ECB and CBC.

# Usage

```
$ cipher.py -i apple.jpg
```

```
$ cipher.py -i cat.jpg
```

```
$ cipher.py -i moon.jpg
```
